# House-Price-Prediction-
The goal for the project is to use machine learning to estimate real estate price based on various features such as location, size , and property condition.

Project Overview 

Objective: Develop a predictive model that accurately estimates the market value of residential properties using historical sales data.
Key Features: Models typically analyze parameters like square footage, number of bedrooms/bathrooms, lot area, year built, and neighborhood location.
Target Variable: The sale price of the house (a continuous numerical value)
